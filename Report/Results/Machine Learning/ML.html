Introduction

In this tutorial, we will analyze a simulated RNA-seq dataset. Our goal is to identify Differentially Expressed Genes (DEGs) between two conditions (Condition A and Condition B), visualize the results, and use both unsupervised and supervised machine learning techniques to classify the samples based on their gene expression profiles.

Step 4: Isolate Differentially Expressed Genes

To reduce noise and dimensionality for our machine learning algorithms, we will subset our normalized count matrix to only include the statistically significant DEGs identified above.

In order to make sure we don’t lose too much information, let’s relax a bit the thresholds for significance

# Strict thresholds: padj < 0.05, log2FC >= 1 (2-fold change)
DEGmat <- normalized_counts[which(res$padj<=0.05 & abs(res$log2FoldChange)>=1),]

# Relaxed thresholds: padj < 0.1, log2FC >= 0.58 (~1.5-fold change)
# Gives more features for ML, at the cost of including noisier genes
DEGmat_relax <- normalized_counts[which(res$padj<=0.1 & abs(res$log2FoldChange)>=0.58),]
